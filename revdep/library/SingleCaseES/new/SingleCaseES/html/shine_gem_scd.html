shine_gem_scd {SingleCaseES}R Documentation

Gradual Effects Model SCD Calculator

Description

Open an interactive tool for calculating the gradual effects model for SCDs.

Usage

shine_gem_scd(browser = TRUE)

Arguments

browser

logical value indicating whether to launch the app in the system's default web-browser. Defaults to TRUE.

Examples

## Not run: 
shine_gem_scd()

## End(Not run)
